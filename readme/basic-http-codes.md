# Basic HTTP Codes

The following HTTP status codes are returned by the WAOne API to indicate whether an API request has been successfully processed or if an error occurred during the request.

| Code | Meaning               |
| ---- | --------------------- |
| 200  | Success               |
| 400  | Bad Request           |
| 401  | Unauthorized          |
| 403  | Forbidden             |
| 404  | Not Found             |
| 500  | Internal Server Error |
