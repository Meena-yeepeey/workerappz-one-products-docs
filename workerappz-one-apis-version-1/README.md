# WorkerAppz One APIs (Version 1)

This section provides detailed documentation for all available **WorkerAppz One API endpoints**. Each API includes its purpose, endpoint information, request parameters, request and sample response, and possible error codes.

These APIs enable integration with the WorkerAppz One system to perform operations such as retrieving request fields, calculating exchange rates, searching banks, creating transactions, and retrieving transaction details.

All API requests must be made using the **base URL** and require authentication using the access token obtained from the **Authentication API**.
